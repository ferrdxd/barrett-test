# Barrett Test Nawasena 🎯

> **Kenali potensimu.** Aplikasi web *Assessment Profil* untuk memfasilitasi proses seleksi dan evaluasi potensi diri.

[![Deployed on Vercel](https://img.shields.io/badge/Deployed_on-Vercel-black?logo=vercel)](https://barrett-test.vercel.app)

## 📖 Tentang Proyek

**Barrett Test Nawasena** adalah platform asesmen digital yang dirancang untuk membantu pengguna mengenali potensi diri mereka melalui pengujian profil. Website ini menyederhanakan proses seleksi awal dengan antarmuka yang ramah pengguna. Peserta hanya perlu memasukkan data diri dasar (Nama Lengkap dan Kelas) untuk langsung memulai sesi asesmen.

🔗 **Live Demo / Kunjungi Website:** [https://barrett-test.vercel.app](https://barrett-test.vercel.app)

## ✨ Fitur Utama

- **Form Identitas Cepat:** Input data diri (Nama Lengkap dan Kelas) yang praktis tanpa perlu registrasi akun yang rumit.
- **Asesmen Interaktif:** Alur tes yang terstruktur untuk mengukur dan mengenali potensi peserta.
- **UI/UX Responsif:** Desain antarmuka yang rapi dan dapat diakses dengan optimal melalui perangkat PC/Laptop maupun Smartphone.
- **Performa Cepat:** Deployment yang dioptimalkan menggunakan Vercel.

## 🛠️ Teknologi yang Digunakan

*(Catatan: Silakan sesuaikan bagian ini dengan framework/library asli yang Anda gunakan di *source code* Anda)*
